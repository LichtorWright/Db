# Db (D-Flat) — The Zero-Overhead Ergonomic Systems Language

Db is a high-performance, bare-metal systems language designed for absolute speed, flat memory efficiency, and fluid developer ergonomics.

It does not rely on a heavy runtime, virtual machines, or garbage collection. Instead, Db acts as a highly expressive frontend language that compiles directly down to optimized, bare-metal source code (like C++) via the **taste** universal compiler engine.

## The Philosophy: Ergonomics Without Compromise

In traditional software engineering, you are forced to choose:
- **High-Level Control Flow & Ergonomics:** Languages like C# or Python give you clean syntax, but saddle you with runtime bloat, heap allocation dependency, and garbage collection pauses.
- **Low-Level Performance:** Languages like C++ give you raw hardware speed, but trap you in dense, verbose boilerplate and complex header/dependency management.

**Db eliminates the compromise.** It gives you the clean layout and expressive vocabulary of a modern scripting language while compiling down to blistering fast, cache-friendly, hardware-aligned logic.
